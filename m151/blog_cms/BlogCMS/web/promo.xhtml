Introduction

Welcome to the Documentation of Blog/CMS. Here you should find everything you need to know to use this platform.

The developer manual doesn't exists yet but maybe you can understand the code without it.

Page: Login

top

The Login Page is the first Page you will see. It is not different from other login sites of other sites. On one side you can see a form for a login and on the other for signup.

The Login form is for Users who already have an account. (If you don't have an account, pleas use the signup form). Enter your username or email and your password. You'll get an error message if your password is incorrect!

With the signup form you can create a new account. Fill in your username, choose a password, repeat it so you can be sure you typed it correct. Then fill in your email and choose your gender. If you don't want to specify your gender for whatever reason you can choose Other.

If you don't get an error message you should now see the main page!

Page: Main

top

The Main-Page is the page with the most functionalities. On the left you see a list with all posts. In addition to the post title there is also the author and the release date of the post. If you use the search input you can filter the post; it searches in the post content and the post title.

On the right you can see the whole post: There is following information

  • Post Title
  • Post Content
  • Date when the post was released
  • The bloggers username
And following functionalities:
  • Pin the post to your pinwall
  • Share the post to:
    • Facebook
    • Twitter
    • E-Mail

If you are in the content editor mode, you can also create new content if you click the new Post button. If you look at a post you've created, you can also edit the text with the edit post button

On the bottom of the page, you can see the comment section. This part should also be self explaining as it is exactly the same as in other tools and sites. Write a comment and press the comment button. It maybe takes a while to appear because of the database structure.
